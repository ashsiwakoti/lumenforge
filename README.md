# LumenForge

Interactive Monte Carlo path tracer for physically-based rendering — being built end-to-end by the [Rasputin](https://github.com/ashsiwakoti/rasputin) multi-agent engine on 2026-04-29.

## Run #1 scope

Brute-force path tracer + tone mapping. 8 features (F-001 through F-007 + F-009). No acceleration structure, no frontend. See `shared/design_schema.json` in the engine repo for the input contract.

This is an n=2 generalization test for the engine — different domain from prior builds (VoxAlign / Chronolog / Snip).

Commits with `[ARCHITECT]` and `[ENGINEER]` prefixes are auto-generated by the multi-agent engine; the manual finale commit (if any) updates the KNOWN_LIMITATIONS doc post-build.
